get_master_document_name(package=None)[source]

Find the first .rst file Sphinx should point to.

Ideally this shouldn’t be necessary but it looks like, while building rez_sphinx in Python 2 + sphinx-rtd-theme, the name of this file gets swapped to “contents.rst”.

So we name it “index.rst”, for consistency.
