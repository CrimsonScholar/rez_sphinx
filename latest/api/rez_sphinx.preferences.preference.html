get_master_document_name()[source]¶

Find the first .rst file Sphinx should point to.

Ideally this shouldn’t be necessary but it looks like, while building User Documentation in Python 2 + sphinx-rtd-theme, the name of this file gets swapped to “contents.rst”.

So we name it “index.rst”, for consistency.

References

https://github.com/readthedocs/readthedocs.org/issues/2569#issuecomment-270577290

Returns

The base name, without file extension, of the “entry point” for Sphinx.

Return type

str
