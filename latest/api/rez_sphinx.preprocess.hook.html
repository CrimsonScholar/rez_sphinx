preprocess_help(package_source_root, help_)[source]

Get auto-generated help attributes for a Rez source package.

If the source Rez package at package_source_root has no Sphinx conf.py, this function returns nothing.

Parameters

  • package_source_root (str) – An absolute path to a directory on disk where some source Rez package lives.

  • help (list[str] or str or None) – The found Rez package help, if any.

Raises

ValueError – If package_source_root isn’t in a Rez package.

Returns

The original help_ plus any auto-generated help entries.

Return type

list[list[str, str]]
