rez_sphinx.preferences.preference_environment

A preference module which specializes in parsing environment variables.

get_overrides(schema)[source]

Find every override which comes from the user’s environment variables.

Parameters

schema (dict[str, dict[str, object] or object]) – A nested dictionary used to discover and read environment variable data.

Returns

All found overrides, if any. Most of the time, this will be an empty dict. Most people probably won’t set overrides via environment variables.

Return type

dict[str, dict[str, object] or object]

get_paths(schema)[source]

Get every preference path that depends on a schema.

Parameters

schema (dict[str, dict[str, object] or object]) – A nested dictionary.

Returns

Every preference path, split into two sets. The first set contains “normal” preference paths which have no dynamic content. The second set has at least some content that must be provided by the user in order to fully resolve.

Return type

tuple[set[str], set[str]]
