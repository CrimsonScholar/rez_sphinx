rez_sphinx.commands.publish_run¶

The module which handles the rez_sphinx publish run command.

build_documentation(directory)[source]¶

Build all User Documentation registered documentation at directory.

Parameters

directory (str) – The absolute path to a folder on disk. This folder should be a sub-folder inside of a Rez package to the root of the Rez package.

Raises

NoDocumentationFound – If any found tests attributes ran, succeeded to run, but no destination documentation could be found.

Returns

The root directories on-disk where all documentation was generated into. For typical users, this list contains only one value.

Return type

list[str]

get_all_publishers(package)[source]¶

Find every registered publisher for package.

Every found publisher will be auto-connected to the remote git repository that they’re associated with (e.g. GitHub).

Parameters

package (rez.packages.Package) – A package which may contain extra configuration overrides. Any override not found will be retrieved globally.

Returns

Every registered publish method.

Return type

list[Publisher]
