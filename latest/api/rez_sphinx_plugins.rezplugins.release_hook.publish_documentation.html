rez_sphinx_plugins.rezplugins.release_hook.publish_documentation

The main module for defining pre-release and post-release behavior.

See PublishDocumentation for details.

class PublishDocumentation(source_path)[source]

Bases: rez.release_hook.ReleaseHook

The class which runs pre-release / post-release methods.

  • On pre-release: Add help keys (See Adding Documentation Automatically).

  • On post-release: Publish the documentation to remote (See rez_sphinx publish run).
