rez_sphinx.core.constant

Any variables needed by other modules within rez_sphinx.

REZ_SPHINX_OBJECTS_INV

A key needed by Sphinx, internally, in order to load objects.inv files. This key is auto-added to Rez packages on-build.

Type

str

ROOT_REPLACEMENT

A string to search for and replace while modifying the user’s help entries. This later may be replaced with a website address or a local Rez package path so users can find the documentation more easily.

Type

str
