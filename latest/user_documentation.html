User Documentation

rez_sphinx is the bridge between Rez and Sphinx. It takes all of the Sphinx’s great documentation-generation features and tightly integrates it with Rez’s excellent environment management system. As a result, you get documentation which is auto-configurable, auto-builds, and auto-links across projects.

These documentation pages are resist link rot, ensuring the pages you spent time writing are built to last.

Check out the Beginner Topics below and, after that, consider reading through All Commands.
