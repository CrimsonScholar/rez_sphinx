get_mode_by_name(name)[source]¶

Find a callable function matching name.

Parameters

name (str) – A registered possibility, e.g. "config" or "guess".

Raises

ValueError – If name is not a registered command.

Returns

A function which takes a list of Rez packages, for example, and returns a list with each depth’s packages, starting from depth=0 and ascending order.

Return type

callable[list[rez.packages.Package] -> list[list[rez.packages.Package]]]
