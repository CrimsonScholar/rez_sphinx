rez_sphinx.api¶

All public-facing commands which User Documentation allows other packages to run.

bootstrap(data, package=None, skip=frozenset({}))[source]¶

Gather Rez package information for Sphinx.

This data is returned and used to auto-fill-out the values for Sphinx conf.py.

Parameters

  • data (dict[str, object]) – Incoming locals() data from an existing Sphinx conf.py.

  • package (rez.packages.Package, optional) – The package to query from and extend data with extra information. If no package is given, the package is determined automatically from the caller’s site.

  • skip (set[str], optional) – Any variable which you do not want rez_sphinx build run to set. This includes both the built-in variables, such as "intersphinx_mapping" and even dynamic variables set using sphinx_conf_overrides

Returns

All Rez data to send to the Sphinx conf.py.

Return type

dict[str, object]

package_preprocess_function(this, data)[source]¶

Replace the package help in data with auto-found Sphinx documentation.

If no rez_sphinx tags are found, this function will exit early and do nothing.

Parameters

  • this (rez.packages.Package) – The installed (built) Rez package. This package is mostly read-only.

  • data (dict[str, object]) – The contents of this. Changing this instance will have an effect on the Rez package which is written to-disk.
