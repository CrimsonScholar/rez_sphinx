run(namespace, modify=True)[source]

Run the selected subparser.

Parameters

  • namespace (argparse.Namespace) – The parsed user content. It should have a callable method called “execute” which takes namespace as its only argument.

  • modify (bool, optional) – If True, run extra calls directly modifying namespace before calling the main execution function. If False, just take namespace directly as-is.
