rez_sphinx.commands.initer

The module which handles the rez_sphinx init command.

init(package, quick_start_options=(), skip_existing=False, verbose=False)[source]

Connect a Rez source package to rez_sphinx.

Warning

This function generates many files directly underneath package as well as changes your source Rez package.py file. Save your work first before running this function!

Parameters

  • package (rez.packages.Package) – The Rez package to apply documentation onto.

  • quick_start_options (list[str], optional) – User-provided arguments to consider while resolving sphinx-quickstart values.

  • skip_existing (bool, optional) – If False, check for documentation and error if existing documentation is found. If True, instead of erroring, do nothing and return early.

  • verbose (bool, optional) – If True, show sphinx-quickstart output.
