rez_sphinx.core.path_control

Miscellaneous functions which make working with file paths easier.

add_gitignore(directory)[source]

Add a .gitignore into directory which ignores all folder contents.

No matter what files or folders are added under directory, they will be ignored by git.

Parameters

directory (str) – An absolute or relative path to a directory on-disk.

clear_directory(directory)[source]

Delete the contents of directory without deleting directory itself.

Parameters

directory (str) – An absolute or relative path to a directory on-disk.

Raises

IOError – If directory does not exist.

expand_path(path)[source]

Convert path from a relative, possibly symlinked location to a real path.

Note

path does not need to exist as a file on-disk.

Parameters

path (str) – It could be “.”, indicating the current directory. Or “../” meaning the parent directory, or an already absolute path.

Returns

The expanded path.

Return type

str

get_installed_root(name)[source]

Find the path on-disk where the resolved name Rez package lives.

Parameters

name (str) – A Rez package name. e.g. “python”, “Sphinx”, etc.

Raises

EnvironmentError – If name has no registered directory.

Returns

Some directory on-disk.

Return type

str

to_environment_name(name)[source]

Change name from a Rez package family name to is env-var equivalent.

Parameters

name (str) – A Rez package name. e.g. "python", "foo.bar", etc.

Returns

The converted name. e.g. "PYTHON", "FOO_BAR", etc.

Return type

str
