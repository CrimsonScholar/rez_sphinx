rez_sphinx.core.environment

Miscellaneous functions for querying the user’s environment.

get_all_repository_uris(package)[source]

Find the location where package’s documentation will be built to.

It’s expected that the destination will be some network URL but this function could also return an absolute / relative path on-disk somewhere, potentially.

Parameters

package (rez.packages.Package) – The source or installed Rez package to query from.

Raises

PluginConfigurationError – If an expected plug-in URL could not be queried.

Returns

The found destination.

Return type

str
